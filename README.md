# Пример использования классов "Автомобиль" и "Велосипед"
# Описание
В этом коде представлены два класса: "Автомобиль" и "Велосипед", которые используются для моделирования транспортных средств. Класс "Автомобиль" представляет собой 
автомобиль с основными характеристиками, такими как марка, модель, максимальная скорость и количество дверей. Класс "Велосипед" описывает велосипед с характеристиками, 
такими как тип рамы и количество педалей. Оба класса наследуют общие свойства и методы от класса "Транспортное средство".
